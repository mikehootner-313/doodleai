Accessibility at DoodleAI

DoodleAI is designed to be usable and enjoyable by everyone — regardless of ability, age, or the tools they use to browse the web.

🔊 Screen Reader Users

DoodleAI works with all major screen readers including NVDA, JAWS, VoiceOver (Apple), and TalkBack (Android).

  • All buttons, inputs, and controls are properly labeled for screen reader announcement
  • The prompt input field is clearly labeled — type your description and press Enter or the Create button
  • Status messages are announced as the drawing progresses so you know what's happening
  • The "story" text below each drawing is fully readable text that describes what's being drawn
  • When a drawing is complete, a message announces it is ready to download
  • The pricing modal is fully navigable by keyboard and announced by screen readers
VoiceOver tip (Mac/iPhone): Use Cmd+F5 on Mac or triple-click the home button on iPhone to activate VoiceOver. Navigate using Tab to move between controls and Space/Enter to activate buttons.
NVDA/JAWS tip (Windows): Use Tab to navigate between controls. The drawing status area is a live region and will be automatically announced as it updates.

🦻 Deaf & Hard of Hearing Users

DoodleAI is a fully visual and text-based experience — there is no audio content on the site, so you will not miss any information by having sound off or by using the site without hearing.

  • All instructions, prompts, and feedback are displayed as visible text on screen
  • The drawing status messages (e.g. "Sketching outlines…", "Adding shading…") appear visually on the canvas
  • Error messages and success notifications are shown as on-screen text
  • No sound effects or audio cues are required to use any feature

👁️ Low Vision & Blind Users

We have designed the interface to be as usable as possible for people with low or no vision.

  • All text meets WCAG AA contrast ratio standards for readability
  • Font sizes are relative (rem-based) and respect your browser or OS font size preferences
  • You can zoom the page up to 200% without losing functionality
  • The "story" text that appears below each drawing describes what the AI is creating — useful when the visual canvas is not accessible
  • All interactive elements are focusable and have visible focus indicators
  • The site works fully with keyboard-only navigation (Tab, Enter, Space, Escape)
Browser zoom tip: Press Cmd + (Mac) or Ctrl + (Windows) to zoom in. The layout will reflow to stay readable at larger sizes.

⌨️ Keyboard Navigation

Every feature on DoodleAI can be accessed using only a keyboard — no mouse required.

  • Tab — move between buttons, inputs, and links
  • Enter — activate buttons or submit the prompt
  • Space — toggle checkboxes and activate buttons
  • Escape — close the pricing modal
  • Arrow keys — adjust the speed slider
  • Shift+Tab — move backwards through controls

📱 Mobile & Touch Accessibility

DoodleAI works on phones and tablets and supports standard touch accessibility features.

  • Compatible with iOS VoiceOver and Android TalkBack swipe navigation
  • All tap targets are large enough to meet minimum touch target size guidelines
  • The site is fully responsive and works at any screen size from small phones to large monitors
  • Pinch-to-zoom is supported — the site does not block browser zoom

🎨 Reduced Motion

DoodleAI features drawing animations by design — watching the art come to life stroke by stroke is the core experience. However, if you prefer reduced motion:

  • Set the speed slider to Fast or Turbo to minimize animation time
  • If your OS has "Reduce Motion" enabled, we respect that preference where possible
  • The final completed image is always available to download regardless of animation settings
